## Air Combat Prototype (UE5, C++)
# Overview

Developed in Unreal Engine 5 using C++ and Blueprints.
Air combat system prototype with player aircraft and AI enemies.

# Features:
1) Player-controlled aircraft with mouse-based steering
2) Shooting and combat system
3) Health and damage system
4) AI turret with targeting behavior
5) AI aircraft (work in progress)
   
# Systems Architecture:
1) FlightComponent
2) WeaponComponent
3) HealthComponent
   
# AI:
1) Behavior Trees
2) Blackboard system
3) Custom Tasks (movement, targeting)
